1. Before mocking any messages, please replace any "REPLACE_ME" tag with actual value.

For any Receiver, use the name of the component that should receive this message.

For Max/Min, it depends on rules of the component, mostly integers.

For any Date/Time, use corresponding time in milliseconds, such that can be obtained by System.currentTimeMillis()

For any RefreshRate, use **NON-NEGATIVE** time in milliseconds.

For any email recipients, make sure that each email is well-formatted and emails are separated by comma.
